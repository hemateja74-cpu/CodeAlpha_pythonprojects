\# Task 3 - CodeAlpha Chatbot



\## 📌 Project Overview

This project is part of the \*\*CodeAlpha Internship (Task 3)\*\*.  

It is a \*\*rule-based chatbot\*\* built using \*\*Python and Flask\*\*, designed to respond to user queries with predefined answers.  

The chatbot runs on a local web server and provides a simple web interface for interaction.



\## ⚙️ Tech Stack

\- \*\*Python\*\* (Core programming language)

\- \*\*Flask\*\* (Web framework)

\- \*\*HTML/CSS\*\* (Frontend templates)
