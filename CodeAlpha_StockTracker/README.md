# 📈 Stock Portfolio Tracker (Console Version)

A Python program to manage and track your stock portfolio directly from the terminal.  
Users can add stocks, view portfolio summary, and save data to a CSV file.

---

## 🚀 Features
- Add stocks with quantity via terminal input
- View portfolio summary with total investment value
- Save portfolio to `portfolio.csv`
